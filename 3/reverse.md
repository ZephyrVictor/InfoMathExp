# 乘法逆元求解流程 (`inverse`)

- ## 初始化
  - 输入: `a` (元素), `m` (模数)
  - 设置: `b = m`, `x0 = 1`, `x1 = 0`

- ## 处理循环
  - **条件**: 当 `a != 0`
    - ### 处理偶数 (a)
      - **条件**: 当 `(a & 1) == 0`
        - `a >>= 1`
        - 更新 `x0`

    - ### 处理偶数 (b)
      - **条件**: 当 `(b & 1) == 0`
        - `b >>= 1`
        - 更新 `x1`

    - ### 比较和调整
      - **条件**: 如果 `a >= b`
        - 减法操作 `a = addorMinus(a, b)`
        - 更新 `x0 = addorMinus(x0, x1)`
      - **否则**
        - 减法操作 `b = addorMinus(b, a)`
        - 更新 `x1 = addorMinus(x1, x0)`

- ## 检查逆元存在性
  - **条件**: 如果 `b != 1`
    - `return 0` (无逆元)

- ## 返回结果
  - `return x1` (逆元)

- ## 辅助函数 `addorMinus`
  - 执行 GF(2^n) 上的加法/减法 (异或运算)
